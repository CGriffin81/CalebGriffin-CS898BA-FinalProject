# ai_log

Keep AI_Log.md in table form.

## What it does

Make every prompt append one new row at end of AI_Log.md. Keep same columns every time: Date/Time, AI Tool, Prompt Type, Full Prompt, Result Synopsis, Design/Code Changes.

## How to use

Use this skill whenever user ask to log prompt history or update AI_Log.md.

## Rule

Always append new entry to bottom of table. Never switch back to header-style entries.
